1)Online Shopping Bill Generator

Create a Java program that:

Takes customer name
Takes product name
Takes product price
Takes quantity
Calculate:
Total bill amount
Output Example:
Customer Name: John
Product: Headphones
Price: 1500
Quantity: 2
Total Bill: 3000

2)Student Result Analyzer

Create a Java program that:

Takes student name
Takes marks of 3 subjects
Calculate:
Total marks
Average marks
Bonus:

Print whether average is greater than 50 using comparison operator.

3)Travel Expense Estimator

Create a Java program that takes:

Travel distance (in KM)
Petrol price per liter
Vehicle mileage
Calculate:
Petrol needed
Total travel cost

Hint:

Use division and multiplication operators.

4)ATM Information System

Create a Java program that:

Takes account holder name
Account balance
Withdrawal amount
Calculate:
Remaining balance
Bonus:

Use comparison operator to check:
balance > withdrawal
