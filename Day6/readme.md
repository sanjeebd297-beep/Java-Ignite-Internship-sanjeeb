2)Treasure Chest Game

Problem Statement


Create a class TreasureChest.

Variables

playerName

coinsCollected

reward

Requirement

Coins >= 1000 → Diamond Sword

Coins >= 500  → Gold Sword

Coins >= 100  → Iron Sword

Else          → Wooden Sword

Constructor should decide reward.

Hint - 

Think like a game developer.

Player collects coins.

Constructor unlocks reward automatically.
